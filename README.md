# 安静位 (Resting Position) 計算ツール

肩関節周囲炎の患者さんの最大外転角度から、介入時の最適な角度（内外旋中間位）を計算するツールです。

## 概要

拘縮のある患者さんの肩甲骨面上での最大挙上可動域を入力すると、関節包が最もリラックスした位置（安静位）を自動計算します。

## 計算式

Ar-Tyan Hsuらの研究(2002)に基づく回帰式を使用しています：

**Y = 0.607X - 13.120**

- **Y**: 予測される安静位（度）
- **X**: 最大外転可動域（度）

## 使い方

1. スライダーを動かして、患者さんの最大外転角度を入力
2. 左側のグラフィックで角度を視覚的に確認
3. 右側のパネルで推奨される介入角度を確認

## 参考文献

- Ar-Tyan Hsu, et al: Determining the resting position of the glenohumeral joint: a cadaver study. J Orthop Sports Phys Ther. 2002 Dec;32(12):605-12.
